技能树：内功与招式

计算机专业的两大基石，缺一不可。

🧠 内功心法 (数学基础)

很多人问：“高数买菜又用不到，为什么要学？”
真相：它们是你未来解锁高级技能的前置条件。

  • 即使听不懂，也要硬着头皮刷题，保持“题感”。
  • 现在的“债”，大三学机器学习时都要加倍还。

⚔️ 招式修炼 (编程语言)

核心法则：Talk is cheap, show me the code.

main.cpp ● Error Found 
#include <iostream>
using namespace std;

int main() {
  int* ptr = nullptr;
  // 致命错误：写入空指针
  *ptr = 2077; 
  cout << "Val: " << *ptr;
  return 0;
}
> Segmentation fault (core dumped)
❌ 误区: 背代码、只看书不动手。
✅ 正解: 启动开发工具，如VS Code。红色的 Error 才是你最好的老师。
🔍 技巧: 多用 AI工具 / Google / Stack Overflow，少用百度。

🎒 装备库：工欲善其事，必先利其器

2025年的学生，要学会拒绝“伪勤奋”，拥抱先进生产力。

VS Code (IDE)

轻量、插件多。拒绝只用记事本或古老的 VC++ 6.0。

推荐插件：
- C/C++
- Prettier
- Error Lens

AI 辅助 (Copilot/DeepSeek)

不要让AI替你写作业，要让它当你的24小时私教。

GitHub

全球最大的代码托管平台（程序员的“军火库”，探索全世界的开源宝藏）。

  • 尽早注册账号
  • 建立 GitHub Profile
  • 学会 `git push` 和 `git pull`
  • 解锁多人联机模式：Git 是团队协作的基石。告别 doc(1), doc(2), doc(3)... 的版本管理噩梦

📜 主线任务 (毕业硬指标)

1. 英语四六级 (CET-4/6)

不要拖到大四！大一/大二记忆力最好的时候一鼓作气拿下。

💡 Tip: 很多大厂招聘和保研都对六级分数有硬性要求 (e.g. > 425/480)。

2. 必修课绩点 (GPA ≥ 2.00)

必修课程累计平均绩点不低于 2.00，否则无法毕业！

⚠️ 警告: 2.00绩点对应百分制约 70分。挂科(< 60分)绩点为0，会严重拉低平均绩点。

3. PU学分 (第二课堂)

毕业门槛之一，拒绝临毕业才疯狂补分。

总实践学时 (150分) 已达成 (486)
校园文化活动 (60分) 96/60
思政社科 (20分) 35/20
公益服务 (10分) 28/10
寒暑假社会实践 (75分) 80/75

🔮 人生航帆：选择你的未来之路

大三之后的分岔路口，没有标准答案。

学术深造

📜 保研 (Rank榜冲刺)

核心：这是一场持续三年的长跑。
不仅看绩点(GPA)，更看科研潜力(Paper)和竞赛奖项。大三下的“夏令营”、“预推免”是关键战役。

⚔️ 考研 (副本Grind)

核心：选择 > 努力。
打破“信息差”比埋头苦读更重要。这是一场孤独的战斗，拼的是心态和自律。

✈️ 出国 (跨服远征)

核心：语言 + 绩点 + 适合的经济支持。
早考雅思/托福。合适想体验不同教育体系或有其他规划的玩家。

职场就业

💼 实习 (实战演练)

核心：尽早接触真实工业界代码。
大二/大三暑假是黄金期。一段大厂/独角兽的实习经历 > 10个课程大作业。

💻 面试 (LeetCode & 八股)

核心：基础不牢，地动山摇。
手撕算法 (LeetCode Hot 100) + 计算机基础 (计网/OS/数据库) 是敲门砖。

🛠️ 技术栈 (武器专精)

核心：T型人才（一专多能）。
选定方向（后端/前端/AI算法/移动端）深耕，用 GitHub 项目证明你的代码能力。

旷野探索

🚀 创业 (建公会)

核心：极强的抗压、xxx。
需要极强的抗压能力和商业嗅觉。适合有改变世界梦想的“疯子”。

💡 独立开发 (独行侠)

核心：产品力 + 运营。
做自己喜欢的产品，养活自己。一人即是一支队伍。

🌏 Gap Year (游历)

核心：寻找自我。
人生不是轨道，而是旷野。停下来思考，也许比盲目奔跑更有意义。

"无论选择哪条路，都没有对错，只要不后悔。"

重要的是找到适合自己的节奏，而不是盲目随大流。

📊 绩点计算规则

⚠️ 重点提醒：

  • 必修课程累计平均绩点 ≥ 2.00 是毕业硬性要求
  • 百分制 70-68分 对应绩点 2.00
  • 低于60分（不及格）绩点为 0，会严重拉低平均绩点
  • 计算公式：平均绩点 = Σ(单门课程学分 × 相应绩点) / Σ 课程学分
