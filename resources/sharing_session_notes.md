### 3.2 ⚔️ 招式修炼 (编程语言)

**核心法则**: `Talk is cheap, show me the code.` (空谈误国，实干兴邦)

#### 实战演练：C++ 指针陷阱
初学者最容易犯的错误——**空指针解引用**。

**❌ 错误示范 (Bad Code)**:
```cpp
#include <iostream>
using namespace std;

int main() {
  int* ptr = nullptr;
  // 致命错误：试图向空指针写入数据
  // 结果：Segmentation fault (core dumped)
  *ptr = 2077; 
  cout << "Val: " << *ptr;
  return 0;
}
```

**✅ 正确姿势 (Good Code)**:
```cpp
#include <iostream>
using namespace std;

int main() {
  int data = 2077;
  int* ptr = &data; // 修复：让指针指向合法的栈内存变量
  
  cout << "Val: " << *ptr;
  return 0;
}
```

**避坑指南**:
1. **拒绝死记硬背**: 不要背代码，不要只看书不动手。
2. **拥抱错误**: 关掉书本，打开 IDE。红色的 Error 报错信息才是你最好的老师。
3. **搜索技巧**: 多用 Google / Stack Overflow / AI 工具，少用百度。

## 4. 🎒 装备库：工欲善其事 (Inventory)

2025年的学生，要学会拒绝“伪勤奋”，拥抱先进生产力。

### 4.1 🛠️ VS Code (IDE)
- **评价**: 宇宙第一编辑器，轻量、插件生态丰富。
- **警告**: 拒绝只用记事本写代码，尽早抛弃古老的 VC++ 6.0。
- **推荐插件**:
  - `C/C++`: 智能提示与调试
  - `Prettier`: 代码格式化
  - `Error Lens`: 让错误提示更直观

### 4.2 🤖 AI 辅助 (Copilot / DeepSeek)
- **定位**: 不要让 AI 替你写作业，要让它当你的 **24小时私教**。
- **Prompt (提示词) 技巧**:
  - ❌ **坏 Prompt**: "帮我写个冒泡排序" -> *结果：你失去了思考能力*
  - ✅ **好 Prompt**: "像给5岁孩子讲故事一样解释这段指针代码" -> *结果：你理解了底层逻辑*

### 4.3 🐙 GitHub (代码托管)
- **定位**: 程序员的“军火库”，全球最大的开源社区。
- **必做任务**:
  - 尽早注册账号，建立自己的 GitHub Profile。
  - 学会 `git push` (上传) 和 `git pull` (拉取)。
  - **解锁多人联机模式**: Git 是团队协作的基石。
  - **版本管理**: 告别 `final_doc.doc`, `final_doc(1).doc`, `final_doc_真的最后版.doc` 的文件管理噩梦。

3.  **低优先级**: 纯理论定义的死记硬背

### 5.2 行动计划
- **时间管理 (倒推法)**: 把最后两周做成甘特图。每天复习哪一章，精确到小时。
- **拒绝自我感动**: 不要“坐在图书馆玩手机”，效率 > 时长。
- **刷题策略**: 刷近3年的真题，搞懂每一道错题。

**心态建设**: 不挂科是底线，高绩点是未来的选择权。
